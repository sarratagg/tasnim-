# Portail Touristique Intelligent - Backend

Node.js + Express backend for the Portail Touristique Intelligent project.

Features:
- Site management (CRUD)
- Hotel booking (hotels, rooms, reservations)
- Tour/circuit booking
- Weather integration (OpenWeatherMap)
- Translation integration (LibreTranslate)
- Recommendations
- JWT auth (users/admin)
- Swagger API docs

Quick start

1. copy `.env.example` to `.env` and fill values
2. npm install
3. npm run dev

API docs will be available at /api/docs when server is running.
